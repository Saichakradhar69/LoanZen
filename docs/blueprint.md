# **App Name**: LoanZen

## Core Features:

- Loan Calculator: A calculator that takes loan parameters, such as interest rate, compounding period, and loan length, to calculate payment amounts and display the full amortization schedule.
- Multi-Loan Dashboard: Allow users to see multiple loans on a dashboard.
- Subscription model: The loan dashboard subscription unlocks value-added services for the user.
- Prepayment Advisor: Provide advice on the user's optimal repayment strategy based on their income and debt situation, recommending a strategy using the 'tool' feature to decide when to incorporate specific savings opportunities or debt management techniques.
- Debt Metrics Overview: Show total debt, weighted average interest rate, and projected payoff date on the dashboard.
- Loan Drill-Down View: Show detailed amortization schedule graph, payment history log, and prepayment simulator for each loan.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust, stability, and financial expertise.
- Background color: Light gray (#F5F5F5), offering a clean and neutral backdrop.
- Accent color: Soft orange (#FFAB40) to highlight key interactive elements and CTAs.
- Body font: 'PT Sans' (sans-serif) for readability and a modern, yet approachable feel.
- Headline font: 'Playfair' (serif) for an elegant, fashionable, high-end feel
- Use crisp, professional icons to represent loan types, repayment options, and key features.
- The landing page uses a split-screen design, and cards are used within the loan dashboards.
